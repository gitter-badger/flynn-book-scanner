### Setup a development environment

Global steps:

* Install nodejs (0.10.x+) + npm
* Install ruby
* `gem install compass`
* `sudo npm install -g grunt-cli`
* `sudp npm install -g bower`

Steps in project directory `./flynn-book-scanner`:

* `npm install`
* `bower install`
* `grunt`
